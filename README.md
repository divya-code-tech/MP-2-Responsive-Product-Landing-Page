# NovaBook Air X1 - Responsive Product Landing Page

## 📌 Project Overview
This is a responsive product landing page built using HTML, CSS, Flexbox, Grid, and Tailwind CSS. It showcases a fictional laptop product called NovaBook Air X1.

## 🚀 Features
- Modern responsive navigation bar
- Hero section with product image and CTA
- Feature cards using Grid and Tailwind CSS
- Benefits section
- Product specifications table
- Customer testimonials
- Pricing section with call-to-action
- Fully responsive design (mobile + tablet + desktop)

## 🛠️ Technologies Used
- HTML5
- CSS3
- Tailwind CSS (CDN)
- Flexbox
- CSS Grid
- Media Queries

## 📱 Responsive Design
The website is fully responsive and works on:
- Mobile devices
- Tablets
- Desktop screens
